# PDF table to csv converter
## This script will convert a a .pdf file to .md if table is contained it will extract it and convert to .csv format for importing to R

You need docling to run this code
best to do is to create a python environment and install docling
via pip then activate it and run this script follow steps below

## *************************************
- python venv -m .doclin
- source .doclin/bin/activate
- pip install docling
